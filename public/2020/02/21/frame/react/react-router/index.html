useHistory

使用这个 hook 可以访问用于导航的 history 记录

useLocation

这个 hook 返回一个 location 对象, 相当于当前的 URL; 你可以把它想像成当 URL 发生变化时 useState 返回了一个新的 location

例: 如果希望每次加载新页面时使用网路分析工具触发新的“页面浏览”的情况, 可以使用这个 hook
