循环结构的应用场景

如果在程序中我们需要重复执行某条或某些指令, 例如用程序控制机器人踢足球, 如果机器人持球且还没有进入射门范围, 那么就要一直发出让机器人向球门方向奔跑的指令; 这个描述中不仅有需要重复的动作, 还有分支结构; 再举一个例子, 比如程序中要实现每隔一秒钟在屏幕上打印一个“hello, world”这样的字符持续一个小时, 那肯定不能将 print('hello, world') 这句代码写3600遍, 因此就需要了解一下循环结构, 有了循环结构就可以轻松控制某件事或某些事重复的发生

在 Python 中构造循环结构有两种做法

  1. for-in 循环
  2. while 循环

for-in 循环

使用场景: 如果明确的知道循环执行的次数或者是要对一个容器进行迭代, 推荐使用 for-in 循环

  • range(101) 产生一个0到100的整数序列
  • range(1, 100) 产生一个1到99的整数序列
  • range(1, 100, 2) 产生一个1到99的奇数序列, 其中的2是步长, 即数值序列的增量
