分支结构的应用场景

迄今为止, Python 代码都是一条一条顺序执行, 这种结构的代码称为顺序结构, 但是仅顺序结构并不能解决所有的问题; 比如游戏会通过分数来判断是否通过, 这就会出现两个分支, 而这两个分支只有一个会被执行, 这就是程序中的分支结构

if 语句的使用

在 Python 中, 要构造分支结构可以使用 if 、elif 和 else 关键字

和 C / C++、Java 等语言不同, Python 中没有花括号来构造代码块而是使用了缩进的方式来设置代码的层次结构, 如果 if 条件成立的情况下需要执行多条语句, 只要保持多条语句具有相同的缩进即可, 换句话说连续的代码如果又保持了相同的缩进, 那么它们属于同一个代码块, 相当于是一个执行整体

如果要构造出更多的分支, 可以使用 if...elif...else 结构; 也可以使用嵌套写法来表示(在 elif 和 else 中再构造新的分支), 即嵌套的分支结构

Python 之禅中有这么一句话 “Flat is better than nested”, 嵌套结构的嵌套层次多了之后会严重影响代码的可读性, 如果可以使用扁平化的结构就不要用嵌套
