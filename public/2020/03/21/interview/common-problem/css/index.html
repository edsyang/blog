水平垂直居中方式(4种)

  • position: absolute; top/right/bottom/left: 0; margin: auto;
  • position: absolute; top/left: 50%; margin-top/left: -50% px;
  • position: absolute; top/left: 50%; transform: translate(-50%, -50%);
  • display: flex; justify-content: center; align-items: center;

清除浮动方法

  • overflow: hidden/auto
  • 在浮动元素后使用一个空元素如<div class="clear"></div>, 并在CSS中赋予 .clear{ clear: both } 属性即可清理浮动

如何让一个元素在页面中消失

  • display: none
  • visibility: hidden
  • opacity: 0

position 定位的理解

  • static 默认值, 正常显示
  • absolute 绝对定位
  • reletive 相对定位
  • fixed 相对浏览器窗口定位
  • inherit 继承父元素定位

css 盒模型

  • 标准盒模型: width(height) = width(height)

  • 怪异盒模型: width(height) = width(height) + padding + border

  • box-sizing:

    • content-box: 宽度和高度分别应用到元素的内容框, 在宽度和高度之外绘制元素的内边距和边框 (标准盒模型)
    • border-box: 为元素设定的宽度和高度决定了元素的边框盒, 通过从已设定的宽度和高度分别减去边框和内边距才能得到内容的宽度和高度 (怪异盒模型)
    • inherit: 规定应从父元素继承 box-sizing 属性的值

对 sass/scss 的理解

  • 为 css 增加了一些编程特性, 不用考虑浏览器兼容性; 让 css 更加简洁, 适应性、可读性更佳, 易于代码维护
  • sass: 以严格缩进语法规则来编写代码的, 不包括大括号和分号
  • scss: 和 css 书写语法类似
