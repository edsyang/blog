快捷键

Editing

Font-Family: Operator Mono

  • Shift + <- and Shift + -> Quickly shrink or expand the current selection.
  • Opt + -> / <- Cursor word left / right move.
  • Cmd + Opt + <- / -> Cursor word left / right select.
  • Cmd + Shift + <- / -> Cursor all-left / all-right side select.
  • Ctrl + - / Ctrl + shift + - go back / go to next editing point.
  • Ctrl + G: Go the specific line.
  • Ctrl + Cmd + G: Insert multiCursor to the selected content in the file.
  • Cmd + Shift + \ go to the next bracket.
  • Control + Shift + K Delete Line.
  • Cmd + Backspace Delete the content all-left side.
  • Cmd + Delete Delete the content all-right side.
  • Opt + Backspace Delete the word all-left side.
  • Opt + Delete Delete the word all-right side.
  • Opt + Cmd + F Replace.

Code Intelligence

  • Alt + Click: Open a new file and show definition
  • Cmd + T: Open all symbols under current workspace
  • Fn + F2: Replace selected symbol in file.
  • Fn + F12: Go to the definition.
  • Fn + Cmd + F12: Go to the implementations.
  • Shift + Cmd + D: Peek definition
  • Shift + Cmd + F12: Peek implementation
  • Cmd + .: Show quick fix dot
  • Shift + Cmd + L Auto fix code format with eslint
  • Shift + Cmd + Alt + L Autofix the code with tslint.

Sys

  • Cmd + O Open .
  • Cmd + B Toggle Sidebar
  • Cmd + P Command panel.
  • Cmd + J Toggle Pane.
  • Opt + Cmd + S Save all active tabs
  • Cmd + K, Cmd + number: cold fold level.
  • Cmd + K, Cmd + J: unfold all.
  • Cmd + Shift + P Command Panel
  • Cmd + Shift + O Search file with symbol
  • Cmd + Shift + T Focus to next terminal
  • Ctrl + Shift + ' Create new integrated terminal.
  • Cmd + K, M Change language mode
  • Cmd + K, D Show diff of the current editor file and saved file
  • Cmd + K, P Copy the path of current working file
  • Cmd + K, R Quick open current active file in Finder
  • Cmd + K, Z Go to the Zen Mode of vscode
  • Cmd + K, cmd + -> switch group focus in editor
  • Cmd + Opt + P: open project manager
  • code . Launching from Command-Line

Search

  • Cmd + P Quickly search for files under working dir.
    • file.exclude will help you exclude the things out of indexing engine.
  • Cmd + Shift + F Quickly search for keywords during whole dir.
