10. express - Node.js框架

express是Node.js的一个框架，用来快速开发服务端应用程序；express可以使服务端的代码组织更清晰、简单

服务端程序可以做：

  1. 创建服务
  2. 监听端口
  3. 提供静态资源服务
  4. 处理ajax接口

express是Node.js的第三方依赖包，需要安装后导入使用

  1. 安装express&body-parser（中间件）
  2. 导入express&body-parser
  3. 使用express创建服务（server）
  4. 使用中间件（middleware）提供静态资源服务

    中间件是一种独立的系统软件或服务程序，分布式应用软件借助这种软件在不同的技术之间共享资源。中间件位于客户机/ 服务器的操作系统之上，管理计算机资源和网络通讯。是连接两个独立应用程序或独立系统的软件。相连接的系统，即使它们具有不同的接口，但通过中间件相互之间仍能交换信息。执行中间件的一个关键途径是信息传递。通过中间件，应用程序可以工作于多平台或OS环境

  5. 配置路由（处理ajax接口），根据不同的pathname做不同的处理，这种机制叫路由
  6. 监听端口号

10.2 express动态路由

  • 路由中的某一部分是可以变的，这一部分需要用:标识，将来这一部分是由客户端传递的；如果没有: 这一部分就是不可以变的；
  • 获取动态路由的值：在express的请求对象中有一个params属性，这个属性保存了动态路由的值
  • param() 方法：app.param() 方法用于拦截所有带id的动态路由，可以访问请求对象req和响应对象res
  • 中间件（middleware）：可以访问请求对象和响应对象的函数；一般中间件都是有固定功能或实现拦截的
