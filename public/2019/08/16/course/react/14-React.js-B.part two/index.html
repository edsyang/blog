13. React轮播图

轮播图：外侧有一个叫 container 的容器，它的宽高只能展示一张图片，溢出的部分是隐藏的，在 container 下面有一个 wrapper 的盒子，这个盒子相对于 container 绝对定位，wrapper 下面包含了很多张图片，这些图片排列在一行，宽高相同。另外还有一个值记录当前 container 展示的图片的索引，当需要切换图片的时候，操作 wrapper 的 left 值，wrapper 的 left 值和图片索引的关系：left = -1 * 索引 * 图宽
